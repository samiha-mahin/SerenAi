## ğŸš€ Getting Started

### Prerequisites

* Node.js >= 18
* MongoDB (local or cloud instance)
* Google Gemini AI API credentials

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/ai-mental-health-platform.git
cd ai-mental-health-platform

# Install dependencies
npm install

# Set environment variables
cp .env.example .env
# Fill in your MongoDB URI, Google Gemini API keys, and NextAuth settings

# Run the development server
npm run dev
```

### Deployment

* **Frontend & Backend**: Vercel or any Node.js-compatible hosting
* **Database**: MongoDB Atlas or self-hosted MongoDB
* **Serverless Workflows**: Deploy Inngest functions to manage AI tasks
